子ども関連アプリは5歳と7歳の子ども2人（2026年7月時点）にテストしてもらっている。ただし対象年齢を5歳・7歳に限定する必要はない。

**Why:** 実ユーザーテストの環境がある一方、対象年齢の設計は柔軟に考えてよい。

**How to apply:** UI・文言の提案では未就学〜小学校低学年が使える水準（ふりがな、読み上げ、タップ領域）を目安にしつつ、対象年齢の幅を狭めすぎない。実機テストの提案時は「お子さんに触ってもらう」検証を選択肢に含めてよい。
